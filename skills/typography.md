# Typography Rules
This document defines  thr typographic system used throughout this applicattion

The system follows the principles influenced by the material design typography scale

## Objectives
Typography provides:
- clear hierarchy
- consistent reading rhythm
- comfortable scanning

## Font Family
primary ui fonts:Roboto

Fall back fonts:
-apple-system
 system-ui
 sans-serif
 Arial
 Inter

 ## Type scale
 use the type scale in the design token

 ##Usage guildline
- page display text should us display large
- section title should use headline large
- button labels should use label large
form labels should use label medium or label large
standard paragraphs and body text should use body large
card titles should use title medium

## Accessibility
- all text size must be measured in rem not pixels
- all heading and display must use the clamp function in css
- low contrast text combination are avoided
- text placed on color surfaces uses the matching "On-*"color token
